A react scroll and intersection observer site built with Next.js.

## Getting Started
- Install all dependencies:
```bash
yarn
# or 
yarn install
```

- Run the development server:

```bash
yarn dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

- Build the app:
```bash
yarn build
```
